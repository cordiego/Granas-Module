Physics Engines

Three Engines, One Module

Each engine operates independently or as part of the unified pipeline. All grounded in peer-reviewed physics — no black-box ML.

ENGINE 1
🔬 Granas Optics — Mie/TMM/AM1.5G
Spectral simulation of granular perovskite thin-films with real absorption, reflection, and EQE
🌀
Mie Scattering
Full angular Mie theory for spherical granules. Computes Qsca, Qabs, Qext, and asymmetry factor g(λ). Wavelength-dependent complex refractive index from built-in material library.
Mie Qext n + ik
📊
Transfer Matrix (TMM)
2×2 characteristic matrix method for multilayer thin-film stacks. Computes R(λ), T(λ), A(λ) across antireflection + active + substrate layers.
TMM R/T/A Multilayer
☀️
AM1.5G Integration
ASTM G-173 solar spectrum integration for short-circuit current density (Jsc). Irradiance-weighted absorption and path length enhancement vs. Yablonovitch 4n² limit.
AM1.5G Jsc 4n²
ENGINE 2
🤖 Bayesian Optimizer — GP + EI
6D Gaussian Process surrogate model for perovskite ink recipe optimization
🧪
6D Parameter Space
Molar concentration (0.8–1.5 M), solvent ratio (DMF:DMSO), spin speed (1–6 kRPM), additive loading (0–5%), anneal temperature (80–200°C), anneal time (5–60 min).
6D GP Matérn 5/2
📈
Physics Objective
Scherrer grain size, defect density, crystallization kinetics, Shockley-Queisser PCE, and T80 stability score. Multi-objective Pareto optional.
PCE Grain T80
🔄
SIBO CLI
Stateless --init / --ask / --tell interface for lab hardware integration. Bash-native. Edge-compatible. < 2s inference latency.
CLI Bash Edge
ENGINE 3
🧠 HJB Controller — Optimal Annealing
Hamilton-Jacobi-Bellman dynamic programming for time-optimal annealing schedules
🧮
Value Function Solver
∂V/∂t + minu{ L(x,u) + ∇V · f(x,u) } = 0. State: (grain_size, defect_density, film_temp). Control: temperature ramp rate (°C/s). 25×20×15 grid with 9 control levels.
HJB DP 3D State
🌡️
Trajectory Output
Optimal temperature profile (°C vs. time), grain growth curve, defect decay trajectory, and total cost metric. Direct feed for lab furnace controllers.
Trajectory °C/s Furnace
